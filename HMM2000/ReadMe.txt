This game is a simple clone of Heroes of Might and Magic in java.

- You have to run the jar archive from the current directory to enable map loading and map saving.

-To compile, generate the jar archive and the javadoc, type in terminal
ant
- To compile, type in terminal
ant compile
- To run, type in terminal
ant run
- To generate the jar archive, type in terminal
ant jar
- To clean up environment, type in terminal
ant clean
- To generate the javadoc, type in terminal (javadoc warnings occur because lawrence libraries sources files missing)
ant javadoc
